# ECOMMERCE-SALES-DASHBOARD
1.Project Title/ Headline :
E-Commerce sales performance dashboard

2.Short Description/ Purpose : 
1-2 sentences explaning what the dashbord does and why it exists.
This interactive dashboard provides a comprehensive overview of e-commerce sales performance,offering key insights into sales trends ands customer behaviour.It visualizes critical metrics such as revenue,profit,quantity sold,and payment methods through dynamic charts and graphs.The dashboard allows for in depth analysis of sales across different customer segments,product categories,payment modes ,and geographical regions.

3.Tech Stack :
List the key technologies used to build the dashboard.
-Power Bi Desktop : Main data visualization platforms used for report creation.
-Power Query : Data transformation and cleaning layer for reshaping and preparing the data.
-DAX(Data Analysis Expression) : Used for calculated measures,dynamic visuals,and conditional logic.
-File Format : .pbix for development and .png for dashboard previews.

4.Data Source :
Source:https://mavenanalytics.io

5.Features / Highlights : 
*Interactive visualization
*Sales trends Analysis
*Customer and Product insights
*Comparative Analysis
